# Persona Blocks

Persona and style wording blocks.

These files may describe tone and role language. They must not decide task routing or permissions.

## Active Templates

- `persona_identity.md` contains the assistant identity sentence.
- `persona_user_profile_header.md`, `persona_profile_field.md`, and `persona_direct_address.md`
  contain user-profile wording.
- `persona_onboarding_header.md`, `persona_bullet.md`, `persona_personality.md`,
  `persona_tone.md`, and `persona_verbosity.md` contain general persona sections.
- `persona_live_tools_header.md`, `persona_live_tool_line.md`, `persona_tool_usage_rules.md`,
  `persona_container_management.md`, `persona_trion_home.md`, and `persona_cron_autonomy.md`
  contain dynamic tool-awareness wording.
- `persona_rules_header.md`, `persona_numbered_rule.md`, and `persona_privacy_header.md`
  contain rule and privacy section wording.
